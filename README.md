# Madco Auth For Laravel

## How To Install
Use This Code On PowerShell Or CMD:
```bath
composer require madcobiz/madco-auth-laravel
```

After Install Use This:
```bath
php artisan vendor:publish madcobiz/madco-auth-laravel
```
